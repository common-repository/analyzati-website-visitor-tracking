== Description ==
The Analyzati plugin is a WordPress plugin that allows you to activate website visitor tracking.

When you install our plugin and activate the 'website tracking', our script will be added inside the head section of your website. This script is necessary to track your website visitors. 

Remember our code weight is less than 1kb of size.

Please feel free to visit our Privacy policy at https://analyzati.com/privacy-policy/ and our Terms of service at https://analyzati.com/terms-conditions/

If you have any questions regarding our product, just contact us visiting our contact page at https://analyzati.com/contact/

== Installation ==
1. Upload the 'Analyzati' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Analyzati' menu to access the settings page.
4. Select "Activate Website Tracking" and save changes.
The custom script will now be added before the head tag on all pages of your WordPress website.

== Frequently Asked Questions ==
= How do I add my custom script? =
After activating the plugin, go to the 'Analyzati' menu. In the settings page, activate the option 'Activate Website Tracking' and save changes.

= What kind of code is added into the head section? =
When you activate the website tracking, our script is added to the head section of your website. If you want to read more, visit our knowledgebase at https://analyzati.com/help/get-started/copy-the-script/
